# Behavioral-Ransomware-Detection-System-
Python, Sysmon, Isolation Forest, LSTM, SHAP, Flask 

– Building a proactive AI-based detection system that identifies multi-stage ransomware activity in the
pre-encryption phase using real-time system telemetry.

– Implemented Isolation Forest for anomaly screening and LSTM/Transformer models for sequential attack pattern
recognition across file I/O, registry, and process chain data.

– Designed an automated containment pipeline using PowerShell and WMI to isolate hosts, terminate malicious
processes, and archive forensic evidence upon detection.
